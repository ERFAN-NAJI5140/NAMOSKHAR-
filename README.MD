# 🔥 NAMOSKHAR Theme

Professional Red & Black Theme for Termux

## Features

- 🔥 Red Neon Theme
- 💀 Hacker Style
- ⚡ Boot Animation
- 🖥 Custom Banner
- 🔋 Battery Info
- 📱 Android Version
- 🌐 IP Address
- 🕒 Time & Date
- 🎨 Custom Prompt

---

## Installation

```bash
pkg update -y

pkg install git -y

git clone https://github.com/ERFAN-NAJI5140/NAMOSKHAR-

cd NAMOSKHAR-

chmod +x install.sh

bash install.sh
```

---

Restart Termux.

Enjoy ❤️
